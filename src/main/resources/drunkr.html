/users/{userId}/bac get

get

Retreive all BAC calculations for the user within the given time range

get /users/{userId}/bac

Retreive all BAC calculations for the user within the given time range

  • Request
  • Response

URI Parameters

  • userId: required (string)

Query Parameters

  • from?: (datetime)

    start range inclusive to get BAC's

    Example:

    2017-12-25T16:17:29.000Z
  • to?: (datetime)

    end range inclusive to get BAC's

    Example:

    2017-12-25T16:17:29.000Z

HTTP status code 200

Body

Media type: application/json

Type: array of object

Items: BAC

  • _id: required (string)
  • userId: required (string)
  • bac: required (number)
  • timestamp: required (datetime)

  • Response

Body

Media type: application/json

Type: array of object

Items: Job

  • _id: required (string)
  • startTime: required (datetime)
  • stopTime: (datetime)

    Datetime for when the job is stopped, if not provided will run until sober.

  • source: required (string)

    Source for where push notifications are sent

  • period: required (number)

    Elapsed time in seconds between each BAC calculation.
